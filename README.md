# FinanceControl

FinanceControl is a personal finance dashboard built to track income, expenses, categories, and monthly balance in a simple and organized interface.

This project was created as a portfolio project to practice front-end development, JavaScript logic, data persistence, and SQL data modeling.

## Overview

The goal of this project is to simulate a real-world financial control tool where users can register transactions, classify them by type and category, and monitor their financial summary through a clean dashboard.

The application currently runs entirely in the browser using localStorage for data persistence.

## Features

- Add income and expense transactions
- Track total balance
- Track total income
- Track total expenses
- Count registered transactions
- Filter transactions by type
- Delete transactions
- Persist data with localStorage
- SQL schema for future database implementation
- Responsive dashboard layout
